Support

Last updated: 23 June 2026

Diagramforce is a free, browser-based tool with no account and no backend.

Help inside the app

Click the Help button in the toolbar for a guided walkthrough of the canvas, stencils, sharing, and Google Drive, and hover any control for a tooltip. The project page on GitHub lists features and known limitations.

Reporting a bug

The best place to report a bug is the GitHub issues tracker - open an issue with what you did, what you expected, and what actually happened, plus a screenshot if it is visual. Please do not paste confidential diagrams; describe the problem, or reproduce it with a small, non-sensitive example.
